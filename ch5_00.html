⚡ 전기화학 반응의 속도 ⚡

제5장: 분자 수준에서 바라본 전극 반응의 동역학

5.1 Arrhenius 방정식과 활성화 에너지

📊 Arrhenius 방정식

k = A exp(-Ea/RT)
k: 속도상수 | A: 빈도인자 | Ea: 활성화 에너지 | R: 기체상수 | T: 온도

🔥 실생활 예시: 냉장고의 과학

왜 음식을 냉장고에 보관할까?

상온(25°C)에서 냉장고(4°C)로 온도를 낮추면 세균 증식 속도가 약 2.4배 감소합니다!

5.1.3 전이상태 이론 (Transition State Theory)

⛰️ 에너지 장벽과 반응 경로

k = (kBT/h) exp(-ΔG‡/RT)
kB: 볼츠만 상수 | h: 플랑크 상수 | ΔG‡: 활성화 자유에너지

반응물 A

초기 상태의 분자들

반응이 시작되기 전의 에너지 상태

활성화 복합체

에너지가 가장 높은 전이상태

옛 결합이 끊어지고 새 결합이 형성되는 순간

생성물 B

최종 상태의 분자들

반응 후의 에너지 상태

활성화 에너지

ΔG‡f: 정반응 활성화 에너지

ΔG‡b: 역반응 활성화 에너지

5.2 Butler-Volmer 방정식

⚡ 전기화학 반응의 에너지 다이어그램

평형 상태 (η = 0):
• 산화와 환원의 속도가 같음
• 순 전류 = 0
• 교환전류밀도 i₀만 흐름

Butler-Volmer 방정식

i = FAk°[COx(0,t)exp(-αFη/RT) - CRed(0,t)exp((1-α)Fη/RT)]

5.3 교환전류밀도 (Exchange Current Density)

🔄 평형에서의 동적 평형

i0 = FAk°COx(1-α)CRedα

평형에서: icat = ian = i0

반응 전극 i0 (A/cm²) 특성
H⁺/H₂ Pt 10⁻³ 매우 빠름 ⚡
Fe³⁺/Fe²⁺ Pt 10⁻² 빠름
H⁺/H₂ Fe 10⁻⁶ 중간
H⁺/H₂ Hg 10⁻¹² 매우 느림 🐌

5.5 Tafel 관계와 선형 영역

📏 작은 과전압: 선형 관계

η = -iRct
Rct = RT/(i0F)

Rct: 전하전달 저항 (|η| ≤ 0.1 V에서 유효)

📊 큰 과전압: Tafel 관계

η = a + b log|i|

a: Tafel 절편 | b: Tafel 기울기 (mV/decade)

b = 2.303RT/(αF) ≈ 118 mV/decade (α=0.5, 25°C)

🔄 물질전달 효과가 중요한 경우

가역적 (Reversible)

i0/il = 0.5

매우 빠른 전하전달

준가역적 (Quasi-reversible)

i0/il = 0.05

중간 속도

비가역적 (Irreversible)

i0/il = 5×10⁻⁴

느린 전하전달

5.7 다양한 전기화학 반응의 특성

⚡ 수소 발생 반응 (HER)

전극 재료 i0 (A/cm²) 과전압 @ 1 mA/cm² 응용
Pt (백금) 10⁻³ ~0 mV 연료전지, 수전해 (고가)
Ni (니켈) 10⁻⁵ ~100 mV 알칼리 전해조 (경제적)
Cu (구리) 10⁻⁷ ~400 mV 제한적 사용
Hg (수은) 10⁻¹² ~1000 mV 불리함 (느림)

💨 산소 발생/환원 반응 (OER/ORR)

산소 환원 (ORR)

반응: O₂ + 4H⁺ + 4e⁻ → 2H₂O

특징: 4전자 반응, 복잡한 메커니즘

최적 촉매: Pt/C

산소 발생 (OER)

반응: 2H₂O → O₂ + 4H⁺ + 4e⁻

특징: HER보다 훨씬 느림

최적 촉매: IrO₂, RuO₂

🔬 최신 연구 동향: 백금 대체 촉매

단원자 촉매 (Single-Atom Catalysts)

  • Fe-N-C: 철 원자가 질소에 배위된 탄소 촉매
  • 성능: Pt와 유사한 ORR 활성
  • 장점: 백금 대비 1/100 가격, 풍부한 원소
  • 응용: 차세대 연료전지, 금속-공기 배터리

📊 교환전류밀도 전체 비교

💡 실용적 의미

배터리 설계: 높은 i0 → 빠른 충방전, 높은 출력밀도

연료전지: ORR의 느린 속도 → 백금 촉매 필수 → 비용 증가

수전해: OER의 높은 과전압 → 에너지 효율 감소 → 촉매 개발 필요

부식 방지: 낮은 i0 → 부식 억제 (예: 스테인리스강)
